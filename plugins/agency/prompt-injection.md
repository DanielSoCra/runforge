# Agency Plugin

This plugin is active. You are working on a client website project managed by the Agency plugin.

When you see `.agency.json` in the workspace root, read it first — it contains the client config, stack choice, enabled features, and which workflow to start from.

All research and brand documents go in `docs/` subdirectories. The Astro project lives at the repo root (`src/`, `public/`, `astro.config.mjs`).
