### Task 2 (the real work) ‚Äî wire the verifier gate at PRE-IMPLEMENT (#774 step 4)
**The precise gap (codex round-2 CRITICAL ‚Äî my earlier framing was wrong):** the **pure evaluator already exists and is complete** ‚Äî `evaluateVerifierGate` (`lane-engine/verifier-gate/evaluate.ts:14-29`) checks verifier **present + runnable + falsifying**. - **Add a pre-implement guard** immediately before `coordinator.implement()` (`phases.ts:1518` ‚Äî codex-confirmed the seam has `registry`, `mainRepoRoot`, `run.deploymentId`, and the classifier fields to resolve the same lane path as integrate).
